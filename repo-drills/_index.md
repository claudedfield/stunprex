## How the library evolves

- One drill per scheduled run. Depth over volume.
- Every drill Evaluator-gated before entry. REVISE loops back to Drill Architect (max 2 revisions per session).
- From Drill 11 onward, Drill Architect identifies the capacity gap most under-served and designs for it.
- This index is updated at the end of every run (new row in the Library table, capacity coverage refresh, conviction coverage refresh).
